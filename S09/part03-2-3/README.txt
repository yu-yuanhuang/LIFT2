莫蘭迪配色改版網站包（以 v9 力導向版本為底）

本包保留：
- 左右並列
- 環狀並列
- 力導向布局
- 力導向「重新布局」按鈕
- 手機RWD / 手機選單收合 / 觸控縮放
- 外部讀取同路徑 network.gexf

莫蘭迪配色：
- 人員節點：灰鼠尾草綠 #AEB9AA
- 機構節點：灰霧藕紫 #C6B6BC
- 連線與箭頭：灰米褐 #B9AFA4

使用方式：
1. 解壓後將 index.html 與 network.gexf 放在同一資料夾
2. 用 HTTP 伺服器開啟，例如：
   python -m http.server 8000
3. 瀏覽器開啟：
   http://localhost:8000
